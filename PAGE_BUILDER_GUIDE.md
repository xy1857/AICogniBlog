# 页面装修应用使用指南

## 快速开始

### 1. 装修你的主页

1. 登录后，点击右上角用户名
2. 选择"页面装修"菜单
3. 从左侧拖拽组件到画布
4. 点击组件进行配置
5. 调整主题颜色和样式

### 2. 保存和发布

点击顶部工具栏的"保存"按钮，你的装修配置会被保存。

### 3. 查看效果

有三种方式查看你的装修效果：

#### 方式一：在装修器中预览
点击"预览"按钮，在弹窗中查看效果

#### 方式二：查看我的主页
点击"查看我的主页"按钮，在新标签页打开你的个人主页

#### 方式三：通过菜单访问
- 点击用户名 → "我的主页"
- 或直接访问 `/home/你的用户名`

## 访问路径

### 个人主页
```
http://localhost:5174/home/你的用户名
```

例如：
- `http://localhost:5174/home/admin`
- `http://localhost:5174/home/zhangsan`

### 分享你的主页
你可以将个人主页链接分享给其他人：
```
https://yourblog.com/home/你的用户名
```

## 功能说明

### 装修器功能

1. **组件库**（左侧）
   - Hero横幅
   - 文章网格
   - 分类导航
   - 友情链接
   - 互动组件

2. **画布区**（中间）
   - 拖拽排序
   - 点击选中
   - 复制/删除组件

3. **配置面板**（右侧）
   - 组件配置：修改组件属性
   - 主题设置：全局颜色和字体

4. **工具栏**（顶部）
   - 撤销/重做
   - 预览
   - 查看我的主页
   - 保存

### 个人主页功能

- 展示你装修的页面
- 支持所有组件的交互功能
- 响应式布局，适配移动端
- 可以分享给其他人访问

## 数据存储

### 当前实现（本地存储）

装修配置保存在浏览器的 localStorage 中：
- `pageConfig`: 你的装修配置
- `pageConfig_用户名`: 用于个人主页访问

**注意**：
- 清除浏览器缓存会丢失配置
- 不同浏览器/设备无法同步
- 仅适合演示和测试

### 未来实现（云端存储）

后端 API 实现后，配置将保存到服务器：
- 多设备同步
- 永久保存
- 支持版本历史
- 支持发布/草稿状态

## 常见问题

### Q: 装修后如何让别人看到？
A: 保存后，分享你的个人主页链接：`/home/你的用户名`

### Q: 可以创建多个页面吗？
A: 当前版本每个用户只能有一个装修页面，未来会支持多页面

### Q: 装修会影响原来的首页吗？
A: 不会，装修页面是独立的个人主页，不影响博客默认首页

### Q: 配置会丢失吗？
A: 当前使用本地存储，清除浏览器缓存会丢失。建议定期导出配置（功能开发中）

### Q: 支持移动端吗？
A: 支持，所有组件都是响应式设计，自动适配移动端
